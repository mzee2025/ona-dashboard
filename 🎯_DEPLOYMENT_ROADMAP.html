1 Download Files

Download all files from this package to a folder on your computer.

Required files:

  • app.py
  • ona_quality_dashboard.py
  • dashboard_config.json
  • requirements.txt
  • render.yaml
  • Procfile
  • .gitignore
2 Initialize Git Repository

Open terminal/command prompt in your project folder and run:

git init
git add .
git commit -m "ONA Dashboard Initial Commit"
3 Push to GitHub

Create a new repository on github.com

Then push your code:

git remote add origin https://github.com/YOUR_USERNAME/ona-dashboard.git
git branch -M main
git push -u origin main
4 Deploy on Render.com
  1. Go to render.com and sign up (free)
  2. Click "New +" ‚Üí "Web Service"
  3. Connect your GitHub repository
  4. Render auto-detects settings from render.yaml
  5. Click "Create Web Service"
  6. Wait 3-5 minutes for deployment ‚è≥
5 üéâ Dashboard is Live!
